# Predator-Recon

This is a simple cyber security recon tool made with Python. 

### What it does:
* It scans networks and ports using Nmap.
* It checks for website vulnerabilities using Nikto.
* It can generate quick Netcat reverse shell payloads.
* It has a simple web UI to see all the data easily.and net cat coomand will be generated you just put in to your kali terminal and run it

### Requirements:
* Linux (Kali Linux recommended)
* Python 3
* Nmap and Nikto installed
